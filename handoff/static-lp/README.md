# Static LP Handoff

このフォルダは、一般的なHTML/CSSサイトとして公開・再利用するための軽量版です。

## 編集箇所
- 文言: `index.html`
- 色・フォント・文字サイズ・余白: `styles.css`
- 画像: `images/`

## 使い方
- 普通のWebサーバーにこのフォルダごとアップロードすれば表示できます。
- WordPress で使う場合は、文言を固定ページに移し、`styles.css` をテーマ側のCSSへ移植するのが安全です。
- Wix / Google Sites / Notion では、このHTMLをそのまま貼るより、構成見本として使う方が現実的です。
